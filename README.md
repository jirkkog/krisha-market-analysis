# 🏠 Анализ рынка недвижимости Алматы (Krisha.kz)

Исследовательский проект по анализу рынка вторичной и первичной 
недвижимости Алматы на основе данных с платформы Krisha.kz.

## Цель проекта

Найти инсайты в данных о продаже квартир и сформулировать 
продуктовые рекомендации для платформы Krisha.kz на основе анализа.

## Ключевые выводы

- **Разрыв между районами — 1.6x**: медианная цена за м² варьируется 
  от 631k ₸ (Турксибский) до 1007k ₸ (Бостандыкский)
- **Тип квартиры не влияет на цену за м²**: медиана одинакова для 
  1, 2 и 3-комнатных (~700k ₸/м²) — ключевой фактор это район
- **Этаж почти не влияет**: первый этаж дешевле среднего лишь на 4.7%, 
  последний — на 5.2% (в отличие от московского рынка где дисконт 15-20%)
- **Турксиб — район активного строительства**: 30% объявлений — 
  новостройки при самых низких ценах
- **10% объявлений переоценены на 68-149%** относительно медианы района

## Стек

- **Python**: requests, BeautifulSoup, pandas, numpy
- **Визуализация**: matplotlib, seaborn
- **SQL**: SQLite через sqlite3 + pandas

## Данные

Собрано парсером с Krisha.kz: **824 объявления** о продаже квартир 
по 8 районам Алматы. После очистки и дедупликации: **612 уникальных 
объявлений**.

**Важное замечание по качеству данных:** в процессе анализа обнаружено 
313 дублирующихся объявлений — Krisha показывает одно объявление 
в нескольких районах одновременно. После дедупликации по listing_id 
осталось 612 записей.

## Продуктовые рекомендации для Krisha.kz

**1. Ценовой бенчмарк для продавца**  
При размещении объявления показывать: "Похожие квартиры в вашем 
районе стоят X–Y млн ₸". Топ-10 переоценённых объявлений превышают 
медиану района на 68–149% — бенчмарк помог бы снизить этот показатель 
и ускорить сделки.

**2. Фильтр "Выгодная цена"**  
Выделять объявления с ценой на 15%+ ниже медианы по району 
специальным бейджем. Увеличит вовлечённость покупателей и ускорит 
продажу недооценённых объектов.

**3. Районный индекс доступности**  
Для каждого района показывать не только медианную цену, но и 
минимальный порог входа — поможет покупателям с ограниченным 
бюджетом быстрее находить подходящие варианты.
